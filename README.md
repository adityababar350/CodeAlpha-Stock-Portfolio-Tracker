 Project Description
This project is a simple **Stock Portfolio Tracker** built using Python.
It calculates the total investment value based on user-input stock quantities
and predefined stock prices.

This project is completed as part of the **Python Programming Internship at CodeAlpha**.

Features
- User inputs stock names and quantities
- Uses a hardcoded dictionary for stock prices
- Calculates total investment value
- Saves portfolio summary to a text file
- Beginner-friendly and console-based

Technologies Used
- Python 3
- Dictionary
- Input / Output
- Basic arithmetic
- File handling
